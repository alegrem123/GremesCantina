# Cantina Gremes — Premium Wine Website Design Specification

**Date:** 2026-07-04  
**Project:** Cantina Gremes Brand Website  
**Design Direction:** Editorial Luxury  

---

## 1. Design System

### 1.1 Color Palette

| Role | Color | Hex |
|------|-------|-----|
| **Primary Dark** | Nero Caldo | `#1a1a1a` |
| **Primary Light** | Bianco Avorio | `#f5f3f0` |
| **Neutral Warm** | Beige Naturale | `#d4ccc4` |
| **Heritage Accent** | Verde Vigneto | `#2d4a3d` |
| **Luxury Accent** | Oro Spento | `#a89968` |
| **Optional Warm** | Rosso Vino | `#6b3d3d` |

### 1.2 Typography

- **H1:** Playfair Display 64–96px, weight 700, line-height 1.1
- **H2:** Playfair Display 48px, weight 700, line-height 1.15
- **H3:** Playfair Display 28px, weight 700, line-height 1.2
- **Body:** Inter 16–18px, weight 400, line-height 1.6

### 1.3 Animations

- **Reveal on scroll:** 600–800ms, fade + 20px slide-up, easing cubic-bezier(0.33, 0.66, 0.66, 1)
- **Parallax:** 30–40% scroll offset, subtle
- **Hover effects:** +2% scale, Oro accent border, 200ms transition

### 1.4 Responsive Breakpoints

- Mobile: <768px
- Tablet: 768–1023px
- Desktop: 1024px+
